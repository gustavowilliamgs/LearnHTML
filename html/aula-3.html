Tags e elementos HTML

Como mencionado anteriormente, HTML é composto por tags e elementos. As tags são usadas para marcar o início e o fim de um elemento. Por exemplo, a tag <p> marca o início de um parágrafo e a tag </p> marca o fim do parágrafo.

Existem muitas tags diferentes em HTML, cada uma com um propósito específico. Algumas das tags mais comuns incluem:

  • <h1> a <h6>: Estas tags são usadas para criar cabeçalhos de diferentes tamanhos. <h1> é o maior e <h6> é o menor.
  • <p>: Esta tag é usada para criar parágrafos.
  • <a>: Esta tag é usada para criar links.
  • <img>: Esta tag é usada para inserir imagens na página.
  • <ul> e <ol>: Estas tags são usadas para criar listas não ordenadas e ordenadas, respectivamente.
  • <li>: Esta tag é usada para criar itens de lista dentro de uma lista não ordenada ou ordenada.

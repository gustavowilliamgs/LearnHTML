Elementos semânticos HTML5

HTML5 introduziu vários novos elementos semânticos que ajudam a descrever o tipo de conteúdo que eles contêm. Alguns desses elementos incluem:

  • <article>: Este elemento representa um conteúdo independente e autônomo, como um post de blog ou uma notícia.
  • <aside>: Este elemento representa um conteúdo relacionado ao conteúdo principal da página, mas que pode ser separado dele, como uma barra lateral ou uma caixa de informações.
  • <details>: Este elemento representa um widget que pode ser usado para revelar informações adicionais ao usuário.
  • <figcaption>: Este elemento representa uma legenda para uma figura.
  • <figure>: Este elemento representa um conteúdo autocontido, como uma imagem ou um diagrama, com uma legenda opcional.
  • <footer>: Este elemento representa o rodapé de um documento ou seção.
  • <header>: Este elemento representa o cabeçalho de um documento ou seção.
  • <main>: Este elemento representa o conteúdo principal de um documento.
  • <mark>: Este elemento representa um texto destacado por algum motivo, como uma palavra-chave em um resultado de pesquisa.
  • <nav>: Este elemento representa uma seção de navegação em um documento, como um menu ou uma barra de navegação.
  • <section>: Este elemento representa uma seção genérica em um documento, como um capítulo ou uma aba.

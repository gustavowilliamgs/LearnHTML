Formatação de texto

Em HTML, você pode usar várias tags para formatar o texto de diferentes maneiras. Algumas das tags mais comuns incluem:

  • <strong> : Esta tag é usada para deixar o texto em negrito.
  • <em>: Esta tag é usada para deixar o texto em itálico.
  • <u>: Esta tag é usada para sublinhar o texto.
  • <s>: Esta tag é usada para riscar o texto.
  • <sup>: Esta tag é usada para criar texto sobrescrito.
  • <sub>: Esta tag é usada para criar texto subscrito.

sobre

O presente trabalho tem como objetivo ensinar os fundamentos básicos da linguagem HTML (Hypertext Markup Language). Desenvolvido como parte integrante da disciplina de Programação Web, este projeto aborda diversos conceitos essenciais para a criação de páginas web. Ele foi elaborado por um grupo de três pessoas, composto por Gustavo William Gonçalves Silva, Emanuelle Moreira de Alcântara e Ester Antunes dos Santos. Neste trabalho, iremos explorar os principais aspectos do HTML, desde a estrutura básica de uma página até a utilização de elementos semânticos.

Este trabalho oferece uma introdução abrangente ao HTML, abordando os principais conceitos e fornecendo exemplos práticos para facilitar o aprendizado. Esperamos que, após estudar este material, os leitores tenham adquirido uma base sólida para começar a criar suas próprias páginas web e explorem ainda mais os recursos dessa poderosa linguagem de marcação. Agradecemos aos membros do grupo, Gustavo William Gonçalves Silva, Emanuelle Moreira de Alcântara e Ester Antunes dos Santos, pela dedicação e colaboração na elaboração deste trabalho.

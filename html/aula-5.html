Listas e tabelas

Em HTML, você pode usar várias tags para criar listas e tabelas. Algumas das tags mais comuns incluem:

  • <ul> e <ol>: Estas tags são usadas para criar listas não ordenadas e ordenadas, respectivamente.
  • <li>: Esta tag é usada para criar itens de lista dentro de uma lista não ordenada ou ordenada.
  • <table>: Esta tag é usada para criar uma tabela.
  • <tr>: Esta tag é usada para criar uma linha dentro de uma tabela.
  • <th>: Esta tag é usada para criar um cabeçalho de coluna dentro de uma linha de tabela.
  • <td>: Esta tag é usada para criar uma célula de dados dentro de uma linha de tabela.
